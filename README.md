# Project Phoenix 🔥

An AI system that simulates thousands of trader personas reacting 
to market news before placing a trade.

## How it works
1. Feed a market news headline
2. System builds a knowledge graph of entities and relationships
3. Generates unique trader personas from the graph
4. Simulates how they interact and influence each other
5. Produces a sentiment report with directional signal

## Stack
Python, Groq, NetworkX
